# laravue-weather-app
Weather app using Laravel as Backend and VueJS as Frontend

# Requirements
- PHP >= 8.0
- [composer](https://getcomposer.org/)

# Setup
```bash
cd larave-backend
composer install
cp .env .env-example
php artisan key:generate
php artisan serve
```
You should be able to access the backend at [localhost:8000](localhost:8000)
